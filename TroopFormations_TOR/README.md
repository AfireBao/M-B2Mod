# Troop Formations TOR

Standalone TOR-compatible fork of **Better Troop Formations** (Vombora / Nexus mod 7225).

TOR replaces `PartyScreen.xml` and comments out the formation picker. This module:

1. Re-injects the formation dropdown above TOR party overlays
2. Saves per-troop formation assignments
3. Forces those assignments during field/siege deployment and reinforcements

## Requirements

- Bannerlord Harmony
- ButterLib
- UIExtenderEx
- TOR_Core (load this mod **after** TOR_Core)

Recommended: [TOR Artillery Patcher](https://steamcommunity.com/sharedfiles/filedetails/?id=3723975221) for hidden formation 11/12 issues (separate bug).

## Usage

1. Open Party screen, select a troop (not a hero)
2. Use the formation banner dropdown (right of the center panel)
3. Assign Ironbreakers / Irondrakes / grenadiers to different formations (I–VIII)
